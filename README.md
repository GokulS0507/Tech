# TechDeals India - Deployment & CMS Guide

Congratulations! You have a production-ready, high-converting affiliate marketing website. Follow these steps to go live and start managing products.

## 🚀 Deployment Steps (Netlify)

1.  **Initialize Git**: 
    - Push this folder to a new repository on GitHub.
2.  **Create Netlify Site**: 
    - Connect your repo to Netlify.
    - Publish directory: `.`
3.  **Enable Netlify Identity**:
    - Enable **Identity** and **Git Gateway** in Netlify settings.
4.  **Admin Panel**: 
    - Access via `/admin` on your deployed site.

## 📁 Folder Structure
- `/admin`: CMS Config
- `/assets`: Styles, JS, Images
- `/content`: Product Data (Markdown)
- `index.html`: Home page
- `*.html`: SEO category pages
